# Nitish Rajguru - Portfolio Website

A modern, responsive portfolio website showcasing mobile and web development expertise, with 4+ years of experience in enterprise-grade Flutter application development.

## Features

- **Modern Design**: Clean, professional design with smooth animations
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Scrolling**: Enhanced navigation with smooth scroll behavior
- **Interactive Elements**: Hover effects, animations, and transitions
- **Performance Optimized**: Fast loading times and optimized code
- **SEO Friendly**: Proper meta tags and semantic HTML structure

## Sections

1. **Hero Section**: Introduction with call-to-action buttons
2. **About**: Personal introduction and statistics
3. **Experience**: Timeline of professional experience
4. **Skills**: Technical skills organized by category
5. **Projects**: Showcase of featured projects
6. **Contact**: Contact form and information

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Inter)

## Setup

1. Clone or download this repository
2. Open `index.html` in a web browser
3. No build process required - it's a static website

## Customization

### Update Personal Information

1. Edit `index.html` to update:
   - Name, title, and description in the hero section
   - About section content
   - Experience timeline
   - Skills and technologies
   - Projects
   - Contact information and social media links

### Update Colors

Edit CSS custom properties in `styles.css`:

```css
:root {
    --primary-color: #6366f1;
    --primary-dark: #4f46e5;
    /* ... other colors ... */
}
```

### Add Your Projects

Update the projects section in `index.html` with your actual project details, images, and links.

### Update Social Media Links

Replace placeholder links in:
- Hero section social icons
- Footer social icons
